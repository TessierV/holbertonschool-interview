# Rotate 2D Matrix

This project contains a function that rotates a given n x n 2D matrix 90 degrees clockwise in place.

## Function Prototype


1. **Run the Program:**
   - Use the following command to run the program with example arguments:
     ``` ./main_0.py
     ```

2. **Result:**
   - Use the following command to run the program with example arguments:
     ```[[7, 4, 1],
        [8, 5, 2],
        [9, 6, 3]]
     ```
